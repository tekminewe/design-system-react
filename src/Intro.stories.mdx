# Wallet Roshi's Design System
Welcome to Wallet Roshi's design system, these components is the guidelines when we build Wallet Roshi. You may visit [Wallet Roshi](https://www.walletroshi.com) to see how the components work together.

## Getting Started
To use this design system, you may install the packages with follow command:
```
yarn add walletroshi-design-system styled-components

// or

npm install walletroshi-design-system styled-components
```

## Using Adel font
In Wallet Roshi, we are using "Adel" font. If you wish to use the same, you may add the following line in your `head` element:
```
<link href="https://fonts.googleapis.com/css?family=Abel&display=swap" rel="stylesheet">
```
