# Typography

## Using Quicksand font
In Wallet Roshi, we are using "Quicksand" font. If you wish to use the same, you may add the following line in your `head` element:
```
<link href="https://fonts.googleapis.com/css?family=Quicksand:300,400,500,600,700&display=swap" rel="stylesheet">
```
